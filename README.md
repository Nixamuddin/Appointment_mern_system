🌟 Salon Appointment System

A full-stack MERN application for booking salon services. Users can browse providers and services, select date and time slots, book appointments, and receive email confirmations. The application also features a calendar view for appointments and an admin panel to manage services, providers, and bookings.

📌 Features
User Features

Browse salon services and providers

Select a service, provider, date, and time slot

Book an appointment with email confirmation

View booked appointments in a calendar view

User authentication (register/login)

Admin Features

Admin dashboard for managing:

Providers

Services

Appointments

View appointment statistics

Manage appointment status (pending, confirmed, completed, cancelled)

Calendar overview of bookings

Technical Features

Frontend: React.js

Backend: Node.js + Express.js

Database: MongoDB

Email Notifications: NodeMailer / SMTP

Authentication: JWT-based auth

Routing & API: REST API endpoints

State Management: React Query for data fetching and caching
